About PhytoSynergyDB

The first curated database dedicated to phytochemical-antibiotic synergies against ESKAPE pathogens.

The Problem

Antimicrobial resistance (AMR) is a growing global health crisis. The World Health Organization has declared AMR one of the top ten threats to global public health. The ESKAPE pathogens — Enterococcus faecium, Staphylococcus aureus, Klebsiella pneumoniae, Acinetobacter baumannii, Pseudomonas aeruginosa, and Enterobacter spp. — are responsible for the majority of hospital-acquired infections and are increasingly resistant to conventional antibiotics. New therapeutic strategies are urgently needed.

Our Approach

PhytoSynergyDB addresses this challenge by systematically curating data from peer-reviewed scientific literature on synergistic interactions between plant-derived phytochemicals and conventional antibiotics. Combination therapy — using a phytochemical to potentiate the activity of an existing antibiotic — represents a promising strategy to overcome resistance. Our database provides a centralized, structured resource to accelerate this research.

Methodology & Data Standards

All data in PhytoSynergyDB is manually extracted from peer-reviewed publications through a rigorous curation process.

Literature Search

Systematic search of PubMed, Scopus, and Web of Science for studies reporting phytochemical-antibiotic combination effects against ESKAPE pathogens using checkerboard assays or similar quantitative methods.

Data Extraction

Manual extraction of MIC values (alone and in combination), FIC indices, synergy interpretations, pathogen strain details, and observed mechanisms of action from each qualifying publication.

FIC Calculation

Fractional Inhibitory Concentration (FIC) index is calculated as: FIC = (MICphyto,combo / MICphyto,alone) + (MICabx,combo / MICabx,alone). Interpretation follows standard thresholds.

Quality Control

Each record links to its original source via DOI, ensuring full traceability and reproducibility. Data undergoes validation before inclusion. Chemical identifiers (PubChem CID, InChI Key, SMILES) ensure compound disambiguation.

Synergy Interpretation Standards

FIC Index Range Interpretation Description
FIC ≤ 0.5 Synergy The combination is significantly more effective than either agent alone
0.5 < FIC ≤ 1.0 Additive The combination effect equals the sum of individual effects
1.0 < FIC ≤ 4.0 Indifference The combination has no significant interaction effect
FIC > 4.0 Antagonism The combination is less effective than individual agents

Programmatic Access

PhytoSynergyDB provides a REST API for programmatic access to the data. This enables integration into computational pipelines, meta-analyses, and automated screening workflows. See the API documentation for endpoints and usage.
