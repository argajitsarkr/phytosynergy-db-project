API Documentation

Programmatic access to PhytoSynergyDB for computational research pipelines.

Overview

PhytoSynergyDB provides a read-only REST API for programmatic access to synergy experiment data. All responses are in JSON format. No authentication is required.

Experiments Endpoint

GET /api/v1/experiments/

Returns a paginated list of synergy experiments with full details.

Query Parameters
Parameter Type Description
querystringText search across phytochemical, antibiotic, and pathogen names
interpretationstringFilter by: Synergy, Additive, Indifference, Antagonism
eskapestringFilter by ESKAPE genus (e.g., Staphylococcus, Pseudomonas)
pathogenintegerFilter by pathogen ID
antibioticintegerFilter by antibiotic ID
mechanismstringText search within mechanism of action
limitintegerResults per page (default: 100, max: 500)
offsetintegerNumber of results to skip (default: 0)
Example Request
GET /api/v1/experiments/?eskape=Staphylococcus&interpretation=Synergy&limit=10
Response Format

Statistics Endpoint

GET /api/v1/statistics/

Returns aggregate statistics about the database.

Response Format
{ "total_experiments": 150, "total_phytochemicals": 45, "total_antibiotics": 22, "total_sources": 38, "total_pathogens": 28, "interpretations": { "synergy": 95, "additive": 30, "indifference": 20, "antagonism": 5 }, "eskape_counts": { "Enterococcus": 12, "Staphylococcus": 48, "Klebsiella": 25, "Acinetobacter": 18, "Pseudomonas": 35, "Enterobacter": 12 } }

Similarity Endpoint

GET /api/v1/similarity/

Ranks database phytochemicals by Tanimoto similarity to a query structure, using 2048-bit Morgan/ECFP4 fingerprints.

Query Parameters
smiles (required) - query structure as a SMILES string
limit (optional) - max hits to return (default 25, capped at 100)
threshold (optional) - minimum Tanimoto similarity, 0.0 - 1.0 (default 0)
Example Request
GET /api/v1/similarity/?smiles=Cc1ccc(C(C)C)cc1O&threshold=0.4
Response Format
{ "query_smiles": "Cc1ccc(C(C)C)cc1O", "fingerprint": "Morgan/ECFP4 (radius 2, 2048-bit)", "metric": "Tanimoto", "threshold": 0.4, "count": 3, "results": [ { "compound_name": "Carvacrol", "pubchem_cid": 10364, "inchi_key": "RECUKUPTGUEGMW-UHFFFAOYSA-N", "smiles": "Cc1ccc(C(C)C)cc1O", "chemical_class": "Phenols", "tanimoto": 0.8421, "experiment_count": 7, "synergy_count": 5 } ] }

Usage Examples

Python
import requests response = requests.get( "https://your-domain.com/api/v1/experiments/", params={"interpretation": "Synergy", "eskape": "Pseudomonas"} ) data = response.json() print(f"Found {data['count']} synergistic pairs")
R
library(httr) library(jsonlite) resp <- GET("https://your-domain.com/api/v1/experiments/", query = list(interpretation = "Synergy")) data <- fromJSON(content(resp, "text")) cat(paste("Found", data$count, "synergistic pairs\n"))
curl
curl "https://your-domain.com/api/v1/experiments/?interpretation=Synergy&limit=5"
